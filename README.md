# daynavi

DayNaviは、日々の予定、タスク・就職活動・様々なイベントを一元管理するためのWebアプリケーションです。
通常のカレンダーのように予定を登録するだけでなく、今日やるべきことや締め切りが近い予定をわかりやすく奉持し、ユーザーが設定した確認時刻に予定を把握しやすくすることを目的としています。

## 開発背景

大学の予定、個人開発、学習、就職活動を並行して進める中で、予定や締め切りが複数のツールに分散しやすい課題があります。
特にインターン応募、ES締め切り、面接に一定、webテストなどは見落とすと機会損失につながるため、予定とタスクを一元管理できるアプリを開発します。

## 使用予定技術
- Python
- Django
- SQLite
- HTML / CSS
- Bootstrap
- Git /GitHub

##実装予定機能
- ユーザー登録
- パスワードリセット
- ログイン / ログアウト
- 今日の予定表示
- 日付ごとの予定表示
- 予定詳細表示
- 予定編集 / 削除
- カテゴリ管理
- 確認時刻の設定
- 就活予定管理
- 1日に一度その日の全体予定のリマインド
